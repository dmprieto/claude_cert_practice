Initialize project
1. npm install @anthropic-ai/sdk
2. Create .env.local file and add a variable called ANTHROPIC_API_KEY and initialize it with you api key
3. npm install dotenv @anthropic-ai/sdk

Start example 'Your first API call'
- npx tsx src\api-call-test.ts

Steps to start example 'Agent loop'
- npx tsx src\agent-loop-test.ts
